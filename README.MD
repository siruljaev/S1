# Репозиторий для домашних заданий (1 семестр)

В этом репозитории будут собраны все домашние задания первого семестра.  
Каждое задание оформляется в отдельной ветке по шаблону `feature/hwX-<название>`.

## Структура
- `main` — основная ветка.
- `feature/hw1-gitactions` — ветка для первого домашнего задания.

## Дополнительно
- В корне проекта есть `.gitignore` с типовыми правилами для Java.
- Преподаватели приглашены для проверки.
